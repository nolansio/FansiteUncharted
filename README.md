# Fansite Uncharted

> Ce projet est un exercice en tout début de notre première année.

## Présentation

L'objectif de ce projet est de créer un site web permettant de partager une œuvre de fiction que vous avez apprécié ;
il est possible de choisir un livre, un film, une série, ou un animé.

Un menu de navigation doit permettre de naviguer de page en page ; ce menu devra être responsive et doit apparaître sur
mobile sous forme d'un burger menu ☰.


## Cloner le dépôt
```bash
git clone https://github.com/nolansio/portfolio.git fansite
cd fansite
```

---

### Prérequis
- PHP

### 1. Installer les dépendances
```bash
composer install
```

### 1. Lancer le fansite
```bash
php -S localhost:5050
```

## Accéder aux services
- **Fansite** : http://localhost:5050/
